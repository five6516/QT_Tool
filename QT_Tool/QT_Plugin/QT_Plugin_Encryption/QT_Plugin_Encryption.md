# 加密插件 (QT_Plugin_Encryption)



## 简介

支持对文件及文件夹的批量加解密，支持对字符串的加解密，目前支持AES解密



## 功能

- 文件加密，可直接拖入界面或按钮点击
- 文件解密，可直接拖入界面或按钮点击
- 文件夹加密，直接拖入界面
- 文件夹解密，直接拖入界面
- 字符串加密，按钮点击
- 字符串解密，按钮点击



## 注意点

加密解密都会覆盖原有文件，请做好备份！
